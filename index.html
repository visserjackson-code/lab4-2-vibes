HOW TO USE — RPN MODE

Unlike a standard calculator, the operator comes last.

  1. Type your first number, press ENTER
  2. Type your second number
  3. Select an operator — it locks in orange
  4. Press ENTER again to compute
  5. Press RESET to start over

Example:  4  ENTER  2  ×  ENTER  →  8
